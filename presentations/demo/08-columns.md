###### Layout · columns

## Flowing text in columns

::: columns 2
Body text can flow across multiple newspaper-style columns with **`::: columns 2`** (or `3`).

This is handy for dense reference slides, glossaries, or long lists that would otherwise run off the bottom of the slide.

- linkify, tables and code all still work
- each column balances automatically
- the rule between columns is just CSS
- :check: nesting other containers works too
:::
